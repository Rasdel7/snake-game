# Snake Game 🎮

Classic Snake game built with HTML5 Canvas
and JavaScript, embedded in a Streamlit app.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- Smooth canvas rendering with glow effects
- Snake eyes that follow movement direction
- 4 difficulty levels (Easy → Pro)
- High score tracking in-session
- On-screen touch buttons + keyboard controls
- WASD and Arrow key support

## Controls
- Arrow keys or WASD to move
- On-screen buttons for mobile

## Tools Used
- Python, Streamlit, HTML5 Canvas, JavaScript

## How to Run Locally
pip install streamlit
streamlit run app.py
